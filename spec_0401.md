## 3. Standards & Objectives

**Standards**: HSA.CED.A.1, HSA.CED.A.2, HSF.IF.C.7.D (+), HSF.BF.B.3
**MP**: MP.1, MP.2, MP.4
**Pacing**: 2 days (1 block)

**Objectives** (from LP):
1. Use inverse variation to write and graph reciprocal functions.
2. Identify the effect of transformations on the parent reciprocal function; define effects of h and k on f(x) = 1/(x-h) + k.

**Essential Understanding**: A reciprocal function models inverse variation — a proportional relationship where when one variable increases, the other decreases.

## 4. Content Strands

### Strand 1: Identifying Inverse Variation
- **Concept**: Determine if a relationship is inverse variation by checking if xy = k (constant)
- **Slides**: 3, 4, 5 (Example 1 + Try It)
- **Quiz item**: LQ1 (find p in table)
- **Assessment items**: TTB1
- **Reteach**: RBU1 (fill table with k=48)

### Strand 2: Writing & Using Inverse Variation Equations
- **Concept**: Given a point (x, y), find k, write xy = k or y = k/x, solve for unknowns
- **Slides**: 6, 7, 8 (Concept box + Example 2 + Try It)
- **Quiz items**: LQ1, LQ2
- **Assessment items**: TTB1
- **Reteach**: RBU1, RBU2, RBU3

### Strand 3: Inverse Variation Applications
- **Concept**: Model real-world problems (string frequency, work rates, etc.) as inverse variation
- **Slides**: 9, 10, 11 (Example 3 + Try It)
- **Quiz item**: LQ3 (students washing a car)
- **Assessment items**: TTB1 (rectangle width/length)

### Strand 4: Reciprocal Function Key Features
- **Concept**: Graph f(x) = 1/x; identify domain, range, horizontal asymptote (y=0), vertical asymptote (x=0), end behavior
- **Slides**: 12, 13, 14 (Example 4 + Try It)
- **Quiz item**: LQ4 (horizontal asymptote of y = 1/(x+5) − 4)
- **Assessment items**: *(no clean core anchor — TTB16 is extension only)*

### Strand 5: Translations of Reciprocal Function
- **Concept**: f(x) = 1/(x-h) + k shifts the parent graph h units horizontally, k units vertically; new asymptotes at x=h, y=k
- **Slides**: 15, 16, 17 (Example 5 + Try It)
- **Quiz items**: LQ4, LQ5 (translated graph equation)
- **Assessment items**: TTB5 (translate y=1/x → y=2+1/(x−5)), TTB14 (translate xy=6)
- **Assessment extension**: TTB16 (asymptotes of y=(−x+3)/(x−8) — requires rewriting as y=−1−5/(x−8))

## 7. Vocabulary (Blooket Source)

| Term | Definition | Source |
|------|-----------|--------|
| asymptote | A line that a graph approaches but never reaches | ML #3, #8; Slides 13 |
| constant of variation | The constant k in an inverse variation; the product of x and y (xy = k) | ML #1, #6; Slide 6 |
| inverse variation | A relationship between two variables where as one increases, the other decreases proportionally; xy = k | ML #2, #5; Slides 3, 6 |
| reciprocal function | A function that maps every non-zero real number to its reciprocal; f(x) = 1/x | ML #4, #7; Slide 12 |

| 5 | xy = k | inverse variation |
