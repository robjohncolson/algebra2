Questions:

1. Short answer
Prompt:
`The chart represents an inverse variation. Find the value of p. x-values: 3, 10, 15, 30. y-values: 5, p, 1, 0.5.`
Accepted answers:
`1.5`
`3/2`
Answer key note:
`k = 15, so p = 15/10 = 1.5.`
Alignment:
`TTB1`

2. Multiple choice
Prompt:
`Write an equation for the inverse variation represented by the table. x-values: -3, -1, 1/2, 2/3. y-values: 4, 12, -24, -18.`
Choices:
- `y = x/(-12)`
- `y = 12/x`
- `y = -x/12`
- `y = -12/x`
Correct answer:
`y = -12/x`
Alignment:
`TTB1`

3. Multiple choice
Prompt:
`Three students can wash a car in 16 minutes. If the time varies inversely with the number of students washing the car, how many minutes will it take two students to complete that same job?`
Choices:
- `20`
- `22`
- `24`
- `26`
Correct answer:
`24`
Alignment:
`TTB1`

4. Short answer
Prompt:
`What is the equation of the horizontal asymptote of the graph y = 1/(x+5) - 4?`
Accepted answers:
`y = -4`
`y=-4`
`-4`
Answer key note:
`For y = a/(x-h) + k, the horizontal asymptote is y = k.`
Alignment:
`TTB16`
Note:
`This is the Lesson 1 extension item that still appears on the verified quiz and Topic 4 assessment map.`

5. Multiple choice
Prompt:
`The graph of y = 1/x is translated 3 units up and 2 units to the left. What is an equation of the translated graph?`
Choices:
- `y = 1/(x+2) + 3`
- `y = 1/(x-2) + 3`
- `y = 1/(x-3) + 2`
- `y = 1/(x+3) - 2`
Correct answer:
`y = 1/(x+2) + 3`
Alignment:
`TTB5`, `TTB14`
